Four Hundred Release 20

What's this?

Four Hundred replaces the original System Shock 2 terrain textures with higher resolution, true-color images (16 times the surface of the original). The mod leaves most model skins untouched. If you look at something in-game and a description string appears in the HUD, you're looking at an object (which is a job for SHTUP); everything else is likely for Four Hundred. Feel free to report inconsistencies, wrong colors, weak textures, mistakes and bugs on SystemShock.org

Installation

This mod requires System Shock 2 to be updated to at least version 2.4 (aka New Dark). If you are using the old version of SHTUP, delete OBJ/XERXES.BIN which is a dirty hack that only made sense back when Patch 2.4 didn't exist.

Since release 18, Four Hundred does no longer contain some textures that should now be handled by SHock Texture UPdate: New Dark (or SHTUP-ND for short). These files can still be obtained from another archive, which can be found on gaarabis.free.fr

You can either install FourHundred as a permanent mod (which doesn't require the use of Blue Mod Manager, but cannot be disabled afterward), or as a mod (which requires Blue Mod Manager). In the first case, the destination directory is "SShock2/DataPermanentMods/", in the second case, the destination will be "SSHock2/DMM/FourHundred/".

What's new since release 19?

  • About 150 new textures (Command, Rickenbaker, Recreation, ...)
  • Supports SCP beta 3
  • New glass reflection for OPS
  • Fixed FLT_3/FLT030 (used to display FLT027 instead)
  • Fixed ENG_1/ENGC02 and ENG_1/ENGC01 (a typo prevented texture replacement)

What's next?

"Body of the Many", "Where am I?", some more unused textures and probably some tweaks. Because there are so few textures left to do, everything up to "Body of the Many" SHOULD be considered done, if you spot low-res textures before that point, feel free to let me know by posting a screencap in the Four Hundred thread (check the first post to ensure it hasn't been reported yet).
