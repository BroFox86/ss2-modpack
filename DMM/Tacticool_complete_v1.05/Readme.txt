this package includes all Tacticool weapon models by Mercurius released so far, complemented with Eldron's psi amp mesh (the player "hand") model and RocketMan's psi amp world model. ready to be used together with ADaoB or SCP gamesys mods (if you want to use it with vanilla SS2, you will have to overwrite the obj folder in the root with the one from the vanilla_pistol folder). to install, unzip the archive to your DataPermMods or DataTempMods folder, or use a mod manager.


Tacticool: https://www.systemshock.org/index.php?topic=691.0
Eldron Psi Amp: https://www.systemshock.org/index.php?topic=82.0
RocketMan Alternative Weapons: https://www.systemshock.org/index.php?topic=679.0
extra credits to Olfred for modifying the shotgun and pistol so they would work with SCP properly.
