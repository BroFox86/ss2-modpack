Four Hundred Release 21U
(UNAUTHORIZED EDITION)

What's this?

Four Hundred replaces the original System Shock 2 terrain textures with higher resolution, true-color images at 400% the resolution of the originals. The mod leaves most object skins untouched. If you look at something in-game and a description string appears in the HUD, you're looking at an object (which is a job for SHTUP); everything else is likely for Four Hundred. Feel free to report inconsistencies, wrong colors, weak textures, mistakes and bugs on SystemShock.org

This version has been modified for use with the System Shock 2 Community Patch (SCP), but also works great without SCP. See the list of improvements and fixes below.

Installation

This mod requires System Shock 2 to be updated to at least version 2.4 (aka NewDark). It should be installed using the Dark Engine Mod Manager. All former methods of mod installation are obsolete and no longer supported.

Since release 18, Four Hundred no longer contains some textures that should now be handled by Shock Texture Upgrade Project - NewDark (SHTUP-ND). These files can still be obtained from another archive, which can be found on gaarabis.free.fr

When using this mod in conjunction with SHTUP, you should be using release SHTUP-ND Beta 2 or higher. Older versions will have compatibility issues.

New in Release 21U

  • Additional SCP Beta 5 (and beyond) support.
  • Various improvements to a few textures to more closely resemble the originals.
  • Reconciles conflicts with some SHTUP textures.
  • Removes alpha channel from textures that don't use it.
  • Palletizes textures that don't benefit from being true-color.
  • Reduces install size by 28 more megabytes.

New in Release 20U

  • Supports SCP Beta 5.
  • Makes ceiling fuel pipes in Engineering have directional flow.
  • Removes reflectivity from Hydro and Ops walls. (This just made the walls look transparent.)
  • Upgrades reflective surfaces to use cubemaps.
  • Improves tiling variation of Med/Sci floor tiles, diamond plate floors, and scuffed yellow stripes.
  • Replaces Rec deck grass with photorealistic grass.
  • Fixes self-illumination of casino and shuttle bay animated lights.
  • Adds upgraded snow texture.
  • Various other improvements and fixes.
  • Removes movie theater screen textures. (SHTUP-ND already upgrades these.)
  • Converts all TGA textures to compressed TGAs or PNGs for much smaller files.
  • Reduces install size by 32 megabytes.
